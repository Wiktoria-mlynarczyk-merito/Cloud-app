# Cloud Task Manager

Autor: Wiktoria Młynarczyk
Nr studenta: 100378

## Opis projektu

Aplikacja do zarządzania zadaniami działająca w architekturze chmurowej.

## Stos technologiczny

Frontend: React 19  
Backend: Node.js 24  
Database: Azure SQL

## Architektura

System działa w architekturze 3-warstwowej:

* Presentation Layer – React
* Application Layer – Node.js API
* Data Layer – Azure SQL Database

## Azure Services

|Komponent|Usługa Azure|
|-|-|
|Frontend|Azure Static Web Apps|
|Backend|Azure App Service|
|Database|Azure SQL Database|
